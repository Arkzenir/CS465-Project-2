The Controls are as follows:
Slider - Rotates the tree
Y-shift - Moves the camera by input amount
Z-shift - Changes the orthogonal view volume to the given amount (inputing 1 will create the default view volume) and can be used as pseudo-zoom

Flat color render - Renders a random tree with flat color
Shaded render - Renders a random tree with smooth shading
Wireframe render - Renders a random tree in wireframe

Reset camera - Resets the camera to the starting settings
